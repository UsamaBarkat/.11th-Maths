## 🚀 How to Use This Repository

### For Self-Study:

1. **Start with a Chapter**
   - Check `chapters-and-progress.md` for learning objectives
   - Review the chapter theory from your textbook

2. **Practice Exercises**
   - Try solving exercise questions on your own first
   - Don't look at solutions immediately

3. **Check Solutions**
   - Navigate to exercise files (e.g., `ex1.1.md`)
   - Compare your solutions with provided answers

4. **Track Progress**
   - Update checkboxes in `chapters-and-progress.md`

5. **Use Additional Resources**
   - Check `resources.md` for video lectures and notes

### For Quick Reference:

- Jump directly to specific exercise solution files
- Use formulas and key concepts from solutions
- Review solved examples before exams
