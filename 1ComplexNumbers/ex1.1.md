## Question 1: Evaluate the following

### (i) i¹⁰ · (-i¹²)

**Solution:**
First, recall that powers of i follow a cycle:
- i¹ = i
- i² = -1
- i³ = -i
- i⁴ = 1

For i¹⁰:
i¹⁰ = i^(4×2 + 2) = (i⁴)² · i² = 1² · (-1) = -1

For -i¹²:
-i¹² = -(i^(4×3)) = -(i⁴)³ = -(1)³ = -1

Therefore:
i¹⁰ · (-i¹²) = (-1) · (-1) = **1**

---

### (ii) (-i)¹⁶ · (-i)⁶

**Solution:**
(-i)¹⁶ = [(-1) · i]¹⁶ = (-1)¹⁶ · i¹⁶ = 1 · i^(4×4) = 1 · (i⁴)⁴ = 1 · 1⁴ = 1

(-i)⁶ = [(-1) · i]⁶ = (-1)⁶ · i⁶ = 1 · i^(4×1 + 2) = 1 · (i⁴) · i² = 1 · 1 · (-1) = -1

Therefore:
(-i)¹⁶ · (-i)⁶ = 1 · (-1) = **-1**

---

### (iii) (i)¹¹ · (-i)¹⁴

**Solution:**
i¹¹ = i^(4×2 + 3) = (i⁴)² · i³ = 1² · (-i) = -i

(-i)¹⁴ = [(-1) · i]¹⁴ = (-1)¹⁴ · i¹⁴ = 1 · i^(4×3 + 2) = 1 · (i⁴)³ · i² = 1 · 1³ · (-1) = -1

Therefore:
(i)¹¹ · (-i)¹⁴ = (-i) · (-1) = **i**

## Question 2

### (i) For what value of n, iⁿ is equal to 1, i, -1 or -i, where 1 ≤ n ≤ 4

**Solution:**
Let's evaluate iⁿ for n = 1, 2, 3, 4:

- n = 1: i¹ = **i**
- n = 2: i² = **-1**
- n = 3: i³ = i² · i = (-1) · i = **-i**
- n = 4: i⁴ = i² · i² = (-1) · (-1) = **1**

**Answer:**
- iⁿ = 1 when **n = 4**
- iⁿ = i when **n = 1**
- iⁿ = -1 when **n = 2**
- iⁿ = -i when **n = 3**

---

### (ii) Show that iⁿ + i^(n+1) + i^(n+2) + i^(n+3) = 0, ∀n ∈ Z

**Solution:**
iⁿ + i^(n+1) + i^(n+2) + i^(n+3)
= iⁿ + iⁿ·i + iⁿ·i² + iⁿ·i³
= iⁿ(1 + i + i² + i³)
= iⁿ(1 + i + (-1) + (-i))
= iⁿ(1 + i - 1 - i)
= iⁿ(0)
= **0**

This is true for all n ∈ Z. **Hence proved.**

## Question 3: Simplify the following

### (i) (-6, 3) · (4, -2)

**Solution:**
Using ordered pair notation where (a, b) represents a + bi:
(-6, 3) = -6 + 3i
(4, -2) = 4 - 2i

(-6 + 3i) · (4 - 2i)
= -6(4) + -6(-2i) + 3i(4) + 3i(-2i)
= -24 + 12i + 12i - 6i²
= -24 + 24i - 6(-1)
= -24 + 24i + 6
= **-18 + 24i** or **(-18, 24)**

---

### (ii) (8, -4) ÷ (-2, 2)

**Solution:**
(8, -4) = 8 - 4i
(-2, 2) = -2 + 2i

(8 - 4i) ÷ (-2 + 2i)
= (8 - 4i) / (-2 + 2i)

Multiply by conjugate:
= (8 - 4i)(-2 - 2i) / [(-2 + 2i)(-2 - 2i)]
= [-16 - 16i + 8i + 8i²] / [4 - 4i²]
= [-16 - 8i + 8(-1)] / [4 - 4(-1)]
= [-16 - 8i - 8] / [4 + 4]
= [-24 - 8i] / 8
= **-3 - i** or **(-3, -1)**

---

### (iii) (2, 3) + (4, 5)

### (iv) (6 + 5i) - (4 + 3i)

### (vii) (7 + 4i) + (8 + 5i)

**Solution:**
(7 + 4i) + (8 + 5i)

## Question 4: Simplify

### (i) (2 - i)⁴

**Solution:**
First find (2 - i)²:
(2 - i)² = 4 - 4i + i² = 4 - 4i - 1 = 3 - 4i

Now find [(2 - i)²]²:
(3 - 4i)² = 9 - 24i + 16i²
= 9 - 24i + 16(-1)
= 9 - 24i - 16
= **-7 - 24i**

---

### (ii) (-1 - i√3)²

**Solution:**
(-1 - i√3)²
= (-1)² + 2(-1)(-i√3) + (-i√3)²
= 1 + 2i√3 + i²·3
= 1 + 2i√3 + (-1)·3
= 1 + 2i√3 - 3
= **-2 + 2i√3**

---

### (iii) (-1 + i√3)²

**Solution:**
(-1 + i√3)²
= (-1)² + 2(-1)(i√3) + (i√3)²
= 1 - 2i√3 + i²·3
= 1 - 2i√3 + (-1)·3
= 1 - 2i√3 - 3
= **-2 - 2i√3**

---

### (iv) (1 + i)³

**Solution:**
First find (1 + i)²:
(1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i

Now multiply by (1 + i):
(1 + i)³ = (2i)(1 + i)
= 2i + 2i²
= 2i + 2(-1)
= **-2 + 2i**

---

### (v) (√2 + i)/(√2 - i)

**Solution:**
Multiply by conjugate:
= (√2 + i)(√2 + i) / [(√2 - i)(√2 + i)]
= (2 + 2i√2 + i²) / (2 - i²)
= (2 + 2i√2 - 1) / (2 + 1)
= (1 + 2i√2) / 3
= **1/3 + (2√2/3)i**

---

### (vi) (1 + i)/(1 - i) · (2 - i)/(2 - i)

**Solution:**
First simplify (1 + i)/(1 - i):
= (1 + i)(1 + i) / [(1 - i)(1 + i)]
= (1 + 2i + i²) / (1 - i²)
= (1 + 2i - 1) / (1 + 1)
= 2i/2 = i

Now multiply by (2 - i)/(2 - i):
= i · (2 - i)/(2 - i)
= i · 1
= **i**

---

### (vii) (2 + i)²/(3 - 4i)

**Solution:**
First find (2 + i)²:
(2 + i)² = 4 + 4i + i² = 4 + 4i - 1 = 3 + 4i

Now divide:
= (3 + 4i)/(3 - 4i)

Multiply by conjugate:
= (3 + 4i)(3 + 4i) / [(3 - 4i)(3 + 4i)]
= (9 + 24i + 16i²) / (9 - 16i²)
= (9 + 24i - 16) / (9 + 16)
= (-7 + 24i) / 25
= **-7/25 + 24i/25**

---

### (viii) 1/(2 - i)²

**Solution:**
First find (2 - i)²:
(2 - i)² = 4 - 4i + i² = 4 - 4i - 1 = 3 - 4i

Now find 1/(3 - 4i):
Multiply by conjugate:
= (3 + 4i) / [(3 - 4i)(3 + 4i)]
= (3 + 4i) / (9 - 16i²)
= (3 + 4i) / (9 + 16)
= (3 + 4i) / 25
= **3/25 + 4i/25**

## Question 5: Show that z = 1 ± i satisfies the equation z² - 2z + 2 = 0

**Solution:**

### For z = 1 + i:
z² = (1 + i)² = 1 + 2i + i² = 1 + 2i - 1 = 2i

Substitute into equation:
z² - 2z + 2
= 2i - 2(1 + i) + 2
= 2i - 2 - 2i + 2
= 0 ✓

### For z = 1 - i:
z² = (1 - i)² = 1 - 2i + i² = 1 - 2i - 1 = -2i

Substitute into equation:
z² - 2z + 2
= -2i - 2(1 - i) + 2
= -2i - 2 + 2i + 2
= 0 ✓

**Hence proved that both z = 1 + i and z = 1 - i satisfy the equation.**

## Question 6: Find the conjugate and absolute value of the following

### (i) 4 + 5i

**Solution:**
Conjugate: **4 - 5i**
Absolute value: |4 + 5i| = √(4² + 5²) = √(16 + 25) = √41 = **√41**

---

### (ii) -1 + 7i

**Solution:**
Conjugate: **-1 - 7i**
Absolute value: |-1 + 7i| = √((-1)² + 7²) = √(1 + 49) = √50 = **5√2**

---

### (iii) √3i

**Solution:**
√3i = 0 + √3i
Conjugate: **0 - √3i = -√3i**
Absolute value: |√3i| = √(0² + (√3)²) = √3 = **√3**

---

### (iv) √7 - 3i

**Solution:**
Conjugate: **√7 + 3i**
Absolute value: |√7 - 3i| = √((√7)² + (-3)²) = √(7 + 9) = √16 = **4**

---

### (v) -3 - 4i

**Solution:**
Conjugate: **-3 + 4i**
Absolute value: |-3 - 4i| = √((-3)² + (-4)²) = √(9 + 16) = √25 = **5**

---

### (vi) (5 - 4i)²

**Solution:**
First simplify:
(5 - 4i)² = 25 - 40i + 16i² = 25 - 40i - 16 = 9 - 40i

Conjugate: **9 + 40i**
Absolute value: |9 - 40i| = √(9² + (-40)²) = √(81 + 1600) = √1681 = **41**

---

### (vii) 2/3 - √(-9/16)

**Solution:**
√(-9/16) = √(9/16 · (-1)) = (3/4)i

So: 2/3 - (3/4)i

Conjugate: **2/3 + (3/4)i**
Absolute value: |2/3 - (3/4)i| = √((2/3)² + (-3/4)²) = √(4/9 + 9/16) = √((64 + 81)/144) = √(145/144) = **√145/12**

---

### (viii) (1 + i)(1 + 2i)/(3 + i)

**Solution:**
First expand numerator:
(1 + i)(1 + 2i) = 1 + 2i + i + 2i² = 1 + 3i - 2 = -1 + 3i

Now divide:
(-1 + 3i)/(3 + i)

Multiply by conjugate:
= (-1 + 3i)(3 - i) / [(3 + i)(3 - i)]
= (-3 + i + 9i - 3i²) / (9 - i²)
= (-3 + 10i + 3) / (9 + 1)
= 10i / 10
= i

Conjugate: **-i**
Absolute value: |i| = √(0² + 1²) = **1**

## Question 9: Solve the following complex equations

### (i) (x, y)(2, 3) = (-4, 7)

**Solution:**
(x + yi)(2 + 3i) = -4 + 7i
2x + 3xi + 2yi + 3yi² = -4 + 7i
2x + 3xi + 2yi - 3y = -4 + 7i
(2x - 3y) + (3x + 2y)i = -4 + 7i

Comparing real and imaginary parts:
2x - 3y = -4 ... (1)
3x + 2y = 7 ... (2)

From (1): 2x = -4 + 3y → x = (-4 + 3y)/2

Substitute in (2):
3[(-4 + 3y)/2] + 2y = 7
(-12 + 9y)/2 + 2y = 7
-12 + 9y + 4y = 14
13y = 26
y = 2

Substitute y = 2 in (1):
2x - 3(2) = -4
2x - 6 = -4
2x = 2
x = 1

**Answer: x = 1, y = 2**

---

### (ii) (x + 3i)² = 2yi

**Solution:**
x² + 6xi + 9i² = 2yi
x² + 6xi - 9 = 2yi
(x² - 9) + 6xi = 0 + 2yi

Comparing real and imaginary parts:
x² - 9 = 0 ... (1)
6x = 2y ... (2)

From (1): x² = 9 → x = ±3

If x = 3: 6(3) = 2y → y = 9
If x = -3: 6(-3) = 2y → y = -9

**Answer: (x = 3, y = 9) or (x = -3, y = -9)**

---

### (iii) (x + 2yi)² = xi

**Solution:**
x² + 4xyi + 4y²i² = xi
x² + 4xyi - 4y² = xi
(x² - 4y²) + 4xyi = 0 + xi

Comparing real and imaginary parts:
x² - 4y² = 0 ... (1)
4xy = x ... (2)

From (2): 4xy - x = 0 → x(4y - 1) = 0
So x = 0 or y = 1/4

**Case 1:** If x = 0
From (1): 0 - 4y² = 0 → y = 0

**Case 2:** If y = 1/4
From (1): x² - 4(1/16) = 0
x² - 1/4 = 0
x² = 1/4
x = ±1/2

**Answer: (x = 0, y = 0) or (x = 1/2, y = 1/4) or (x = -1/2, y = 1/4)**

## Question 11: Find the length of vector representing the complex numbers

The length of a vector representing z = a + bi is |z| = √(a² + b²)

### (i) -5 + 2i

**Solution:**
|-5 + 2i| = √((-5)² + 2²) = √(25 + 4) = √29 = **√29**

---

### (ii) 7/3 + 8i/3

**Solution:**
|7/3 + 8i/3| = √((7/3)² + (8/3)²) = √(49/9 + 64/9) = √(113/9) = **√113/3**

---

### (iii) (1 + i)/√2

**Solution:**
First simplify: (1 + i)/√2 = 1/√2 + i/√2

|(1 + i)/√2| = √((1/√2)² + (1/√2)²) = √(1/2 + 1/2) = √1 = **1**

Alternative method:
|(1 + i)/√2| = |1 + i| / |√2| = √(1² + 1²) / √2 = √2 / √2 = **1**

---

### (iv) (1 + √3i)/2

**Solution:**
|(1 + √3i)/2| = √((1/2)² + (√3/2)²) = √(1/4 + 3/4) = √(4/4) = √1 = **1**
